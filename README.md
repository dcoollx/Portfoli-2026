## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   npm run preview
   ```
## Deployment

This repo is auto deployed using github pages and github actions. To update the main site simply merge into main using a PR.
